# whimsey.github.io

Whimsey is a playful website that gives you a same-day “whimsy” activity to try.  
Pick from categories like indoors, outdoors, crafty, or good deeds, then press the button to get a fun prompt.  
You can mark activities as completed and keep a personal “My Whimsies” history in your browser.

## Run automated tests

```bash
npm test
```
